📌 갈피 for Windows

개인정보처리방침

Privacy Policy · 본 방침은 갈피 Chrome 확장 프로그램에 적용됩니다.

갈피는 수집한 데이터를 외부 서버로 전송하지 않습니다.
모든 데이터는 사용자의 Chrome 계정 내에만 저장됩니다.

1. 수집하는 정보

갈피는 다음 정보를 수집합니다.

  • 탭 URL 및 제목: 사용자가 "저장" 버튼을 누를 때 현재 열린 탭의 URL과 페이지 제목을 저장합니다.
  • 컨텍스트 메타데이터: 사용자가 직접 입력한 이름, 메모, 선택한 색상, 생성·수정 시각을 저장합니다.

갈피는 개인 식별 정보, 브라우징 기록, 쿠키, 위치 정보, 결제 정보를 수집하지 않습니다.

2. 정보의 저장 방법

수집된 모든 데이터는 Chrome Storage Sync API를 통해 사용자의 Google 계정에 암호화되어 저장됩니다. 제3자 서버, 데이터베이스, 분석 도구로 전송되지 않습니다.

Chrome 동기화를 통해 동일 Google 계정의 다른 기기에서도 접근할 수 있으며, 이는 전적으로 Google의 보안 인프라에 의해 관리됩니다.

3. 정보의 이용 목적

수집된 정보는 오직 다음 목적으로만 사용됩니다.

  • 사용자가 저장한 작업 컨텍스트를 표시하고 복원하는 기능 제공

광고, 마케팅, 분석, 제3자 공유 등의 목적으로 사용되지 않습니다.

4. 정보의 공유 및 제공

갈피는 어떠한 경우에도 사용자 데이터를 제3자에게 판매·제공·공유하지 않습니다.

5. 데이터 보관 및 삭제

저장된 컨텍스트는 사용자가 앱 내에서 직접 삭제하거나, Chrome 확장 프로그램을 제거할 때 함께 삭제됩니다. Chrome Storage에 저장된 데이터는 Chrome 설정에서도 직접 삭제할 수 있습니다.

6. 권한 사용 목적

  • tabs: 현재 창에 열려 있는 탭의 URL과 제목을 읽어 저장하기 위해 사용합니다.
  • storage: 컨텍스트 데이터를 Chrome Storage에 저장하고 불러오기 위해 사용합니다.
  • sidePanel: Chrome 사이드패널에 UI를 표시하기 위해 사용합니다.

7. 어린이 개인정보 보호

갈피는 만 13세 미만 어린이를 대상으로 하지 않으며, 어린이의 개인정보를 의도적으로 수집하지 않습니다.

8. 방침 변경

본 개인정보처리방침이 변경될 경우 이 페이지를 통해 공지하며, 변경 일자를 하단에 표시합니다. 중요한 변경 사항이 있을 경우 확장 프로그램 업데이트 노트에도 명시합니다.

9. 문의

개인정보처리방침과 관련된 문의사항은 GitHub Issues를 통해 연락해 주세요.
